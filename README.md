# 🧮 Calculator Web App

## 📌 Description  
The Calculator Web App is a simple, responsive, and interactive web-based application developed using HTML, CSS, and JavaScript. It is designed to perform basic arithmetic operations such as addition, subtraction, multiplication, division, and modulus.  

The application provides a clean and user-friendly interface, allowing users to perform calculations easily using both mouse clicks and keyboard inputs. It dynamically updates the display and ensures smooth functionality for real-time calculations.

---

## 🚀 Features  
- Supports basic arithmetic operations (+, -, ×, ÷, %)  
- Allows decimal number calculations  
- Includes Clear (AC) and Delete (DEL) functions  
- Fully functional keyboard support  
- Responsive and simple user interface  
- Interactive button hover effects  

---

## 🛠️ Technologies Used  
- HTML5  
- CSS3  
- JavaScript

---

## 🎮 How to Use  

### Using Mouse  
- Click numbers to enter values  
- Select an operator (+, -, ×, ÷, %)  
- Press '=' to calculate  
- Use AC to clear all values  
- Use DEL to delete last digit  

### Using Keyboard  
- 0–9 → Enter numbers  
- + - * / % → Operations  
- Enter or = → Calculate  
- Backspace → Delete  
- Esc → Clear  
- . → Decimal point
